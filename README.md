## Como executar

Instale as dependências do projeto, rodando os comandos no diretório raiz

```bash
npm install
# ou
yarn
```

Depois rode o projeto como ambiente de desenvolvimento na porta 3000 com o comando

```bash
npm run dev
# ou
yarn dev
```
